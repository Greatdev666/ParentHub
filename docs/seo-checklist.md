# SEO Checklist

## Per-Page Requirements
- [ ] Unique `<title>` tag (50-60 chars)
- [ ] Meta description (120-160 chars)
- [ ] Canonical URL set
- [ ] Open Graph tags (title, description, image, type)
- [ ] Twitter Card tags
- [ ] JSON-LD structured data (Article, BreadcrumbList, FAQ)
- [ ] Single H1 per page
- [ ] Proper heading hierarchy (H1 > H2 > H3)
- [ ] All images have descriptive alt text
- [ ] Internal links use descriptive anchor text

## Technical SEO
- [ ] Dynamic sitemap.xml with all published URLs
- [ ] robots.txt blocking /studio/ and /api/
- [ ] 301 redirects for changed slugs (managed in Sanity)
- [ ] Core Web Vitals: LCP < 2.5s, CLS < 0.1, FID < 100ms
- [ ] Mobile-responsive design
- [ ] HTTPS everywhere
- [ ] Clean URL structure (/category/subcategory/slug)
